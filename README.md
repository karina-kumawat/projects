# QR Code Generator

A simple and clean QR Code Generator built with HTML, CSS, and JavaScript.

## Features
- Generate a QR code for any text or URL
- Choose between 3 sizes (Small, Medium, Large)
- Download the generated QR code as a PNG image
- Clean and responsive design

## Built With
- HTML5
- CSS3
- JavaScript
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) library

## How to Use
1. Clone or download this repository
2. Open the `index.html` file in your browser
3. Enter any text or URL
4. Click "Generate QR Code"
5. Click "Download QR Code" to save it as an image
